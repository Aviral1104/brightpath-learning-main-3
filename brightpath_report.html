Executive Summary

BrightPath Learning is a full-stack, cloud-hosted Learning Management System (LMS) that connects teachers, students, and parents in a single digital platform. Teachers can build structured courses and MCQ assignments, students can enrol and learn at their own pace with rich media content, and parents can monitor their child's academic progress in real time.

The platform is powered by Firebase (Authentication + Cloud Firestore) for zero-maintenance cloud infrastructure, and built on React 18 + TypeScript + Vite for a fast, type-safe frontend. It is deployed on Vercel with CI/CD via GitHub.

Problem Statement

  • Fragmented communication — teachers, students, and parents use different disconnected tools
  • No real-time visibility — parents cannot see their child's daily academic engagement
  • Infrastructure complexity — traditional SQL LMS platforms require DBA expertise and costly hosting
  • Poor accessibility — most systems are desktop-only and have no text-to-speech or focus aids
  • Lack of engagement tools — no built-in focus timers, forums, or announcement channels

System Architecture

The frontend communicates directly with Firebase services via the Firebase JS SDK — there is no custom backend server. All business logic runs client-side with Firestore acting as the single source of truth. Authentication state is managed by Firebase Auth and observed globally via React Context.

Security Considerations

  • Firebase Authentication manages all credentials — passwords are never stored in Firestore
  • Firebase web config keys are client-safe by design (security is enforced by Firestore Rules, not by hiding keys)
  • Profile photos stored as compressed base64 strings in Firestore — no public Storage bucket required
  • Firestore is currently in Test Mode for development; production deployment should add Security Rules

Recommended Production Firestore Rules

rules_version = '2';
service cloud.firestore {
  match /databases/{database}/documents {
    // Users can only edit their own profile
    match /profiles/{uid} {
      allow read: if request.auth != null;
      allow write: if request.auth.uid == uid;
    }
    // Teachers can only modify their own courses
    match /courses/{courseId} {
      allow read: if request.auth != null;
      allow write: if request.auth.uid == resource.data.teacher_id;
    }
    // Authenticated users can read/write everything else
    match /{document=**} {
      allow read, write: if request.auth != null;
    }
  }
}

Conclusion

BrightPath Learning delivers a complete, production-ready LMS built on modern web technologies. By leveraging Firebase's real-time NoSQL infrastructure and Vercel's zero-configuration hosting, the platform eliminates the traditional complexity of school management software.

The codebase is fully Firebase-native — no SQL schemas, no migrations, no server management — making it trivially easy to scale and maintain. The architecture supports real-time data, Role-Based Access Control (RBAC), rich media content, and a polished, accessible UI that works across all devices and screen sizes.

With 3 distinct dashboards, 13 Firestore collections, and 30+ React components, BrightPath is a comprehensive, extensible foundation for modern school education management.
